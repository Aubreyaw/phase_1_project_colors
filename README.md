
# Color Explorer

Color Explorer is an interactive web application that allows users to browse, preview, and explore a collection of colors and their shades. Users can view a list of colors, select individual swatches to see a larger preview, and explore related shades.

This project is designed to provide a seamless, dynamic user experience for color exploration and showcases effective DOM manipulation and API integration.




## Features

- Fetch and display a list of colors from a local API
- Clickable color swatches to preview a larger version and explore shades
- Dynamic and responsive layout for browsing many colors
- Hover effects for better interactivity and usability
- Automatically previews the first color on page load

## Run local API server on 

```bash
http://localhost:3000/colors.
```
## Navigate to the Project Directory:
```bash
cd phase_1_project_colors  
```

Ensure your local server is running and serving the required API.

Open index.html in your browser to launch the app.
## Usage
The app will automatically fetch colors from the API upon loading.
Browse the horizontal list of swatches in the colorContainer.
Click a swatch to view its larger preview and related shades.
Select a shade to update the preview with its color
